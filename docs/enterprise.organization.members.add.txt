NAME:
   dpg enterprise organization members add - Invite users to the specified enterprise's organization

USAGE:
   dpg enterprise organization members add [command options] [arguments...]

OPTIONS:
   --token value                                  [Required] API token [$DPG_API_TOKEN, $DEPLOYGATE_API_TOKEN]
   --enterprise-name value                        [Required] The name of the target enterprise.
   --organization-name value, --group-name value  The name of the target enterprise's organization
   --username value                               The name of the user
   --help, -h                                     show help (default: false)
