NAME:
   dpg organization team member add - Invite users to the specified team

USAGE:
   dpg organization team member add [command options] [arguments...]

OPTIONS:
   --token value                                  [Required] API token [$DPG_API_TOKEN, $DEPLOYGATE_API_TOKEN]
   --organization-name value, --group-name value  [Required] The name of the organization
   --team-name value                              [Required] The name of the target team
   --username value                               The name of the organization's user
   --help, -h                                     show help (default: false)
