NAME:
   dpg enterprise member remove - Remove users from the specified enterprise

USAGE:
   dpg enterprise member remove [command options] [arguments...]

OPTIONS:
   --token value            [Required] API token [$DPG_API_TOKEN, $DEPLOYGATE_API_TOKEN]
   --enterprise-name value  [Required] The name of the target enterprise.
   --username value         The name of the user
   --help, -h               show help (default: false)
